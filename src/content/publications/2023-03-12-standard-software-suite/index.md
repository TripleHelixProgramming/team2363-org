When setting up new laptops for the team, Triple Helix installs the following suite of programs.

## All laptops

- Clean install of Windows 10 or higher

- [Google Chrome](https://www.google.com/chrome/)

## FRC driver station laptops

Triple Helix strongly endorses WPILib's [Driver Station Best Practices](https://docs.wpilib.org/en/stable/docs/software/driverstation/driver-station-best-practices.html) guide for driver station laptops.

- [NI Game Tools](https://docs.wpilib.org/en/stable/docs/zero-to-robot/step-2/frc-game-tools.html)

- [WPILib](https://docs.wpilib.org/en/stable/docs/zero-to-robot/step-2/wpilib-setup.html)

## FRC software development laptops

All software required for an FRC driver station laptop, plus:

- Development environment essentials

  - [Git](https://git-scm.com/)

  - [Clang](https://clang.llvm.org/), [ClangD](https://clangd.llvm.org/), and [ClangD extension for VSCode](https://marketplace.visualstudio.com/items?itemName=llvm-vs-code-extensions.vscode-clangd)

  - [Visual Studio Community](https://visualstudio.microsoft.com/vs/community/) for C++ simulation

  - [HelixNavigator](https://github.com/TripleHelixProgramming/HelixNavigator)

- Networking utilities

  - [FRC Radio Configuration Utility](https://docs.wpilib.org/en/stable/docs/zero-to-robot/step-3/radio-programming.html) for configuring robot radios for home use

  - [Angry IP Scanner](https://angryip.org/download/#windows)

  - [PuTTY](https://www.putty.org/) (example usage: networking with Beelink Mini PC)

- FRC vendor hardware clients

  - [REV Hardware Client](https://docs.revrobotics.com/sparkmax/rev-hardware-client/getting-started-with-the-rev-hardware-client)

  - CTRE [Phoenix Framework](https://store.ctr-electronics.com/software/) and Phoenix Tuner

- Miscellaneous

  - [CBA](http://www.westmountainradio.com/kb_view_topic.php?id=OT39) battery analysis software

  - [Etcher](https://www.balena.io/etcher) (example usage: burning flash drives for Beelink Mini PC)

## FRC mechanical design / CAD laptops

- [PTC Creo](https://www.ptc.com/en/education/student/first)

- [Lightburn](https://lightburnsoftware.com/)

- [PrusaSlicer](https://www.prusa3d.com/page/prusaslicer_424/)
